# 🚀 PHP Auth API (Login & Signup)

A simple **industrial-level PHP API** for user authentication (Signup & Login) using:

- Core PHP (OOP)
- MySQL
- Composer (PSR-4 Autoloading)
- phpdotenv (.env support)
